Práctica: Ejercicios de la Biblioteca de Plantillas Estándar (STL)

Es hora de sumergirse en la poderosa STL de C++. Estos ejercicios te permitirán trabajar con diferentes contenedores, algoritmos e iteradores, facilitando la gestión de datos en tus programas.

Implementa cada ejercicio en tu entorno de Qt Creator para probar el código.

Ejercicio 1: Uso Básico de `std::vector`

Crea un programa en C++ que utilice `std::vector` para:

  1. Declarar un `std::vector` de enteros llamado `numeros`.
  2. Añadir los números 10, 20, 30, 40, 50 al vector usando `push_back()`.
  3. Imprimir todos los elementos del vector utilizando un bucle basado en rango (`for (int n : numeros)`).
  4. Acceder e imprimir el tercer elemento (índice 2) usando `[]`.
  5. Eliminar el último elemento del vector usando `pop_back()`.
  6. Imprimir el tamaño actual del vector.
Pista (si necesitas ayuda)

Recuerda incluir el encabezado ``.

Ejercicio 2: Uso de `std::map` (Diccionario)

Crea un programa en C++ que utilice `std::map` para almacenar y gestionar información de estudiantes:

  1. Declara un `std::map` donde la clave sea el `std::string` (nombre del estudiante) y el valor sea un `int` (su edad).
  2. Añade los siguientes pares clave-valor:
    • "Alice": 20
    • "Bob": 22
    • "Charlie": 21
  3. Accede e imprime la edad de "Bob".
  4. Modifica la edad de "Alice" a 21.
  5. Intenta añadir un nuevo estudiante "Bob" con edad 23. ¿Qué sucede? (Observa, no es necesario justificar con código).
  6. Itera sobre el `map` e imprime todos los nombres y edades de los estudiantes.
  7. Elimina al estudiante "Charlie".
Pista (si necesitas ayuda)

Recuerda incluir el encabezado ``. Para iterar, puedes usar `for (const auto& par : miMapa)`.

Ejercicio 3: Uso de `std::list` e Iteradores

Crea un programa que gestione una lista de nombres usando `std::list`:

  1. Declara un `std::list` de `std::string` llamado `participantes`.
  2. Añade nombres al principio (`push_front`) y al final (`push_back`): "David", "Eve", "Frank" (al final); "Grace" (al principio).
  3. Imprime todos los nombres en la lista usando un iterador (bucle `for` con `begin()` y `end()`).
  4. Usa un iterador para encontrar y eliminar el nombre "Eve".
  5. Imprime la lista nuevamente para verificar la eliminación.
Pista (si necesitas ayuda)

Para eliminar con un iterador: `it = miLista.erase(it);` (esto actualiza `it` al siguiente elemento válido).

Ejercicio 4: Aplicación de Algoritmos STL

Crea un `std::vector` de enteros con valores aleatorios (ej. 10 números entre 1 y 100). Luego, utiliza algoritmos STL para:

  1. Ordenar el vector de forma ascendente.
  2. Buscar si el número `50` está presente en el vector. Imprime un mensaje indicando si se encontró o no.
  3. Contar cuántas veces aparece el número `25` en el vector (asegúrate de que aparezca algunas veces o no, para probar el conteo).
  4. Utilizar `std::for_each` para imprimir cada elemento del vector multiplicado por 2.
Pista (si necesitas ayuda)

Necesitarás incluir `` y `` (para generar números aleatorios si lo deseas, o simplemente usar un `for` para rellenarlo). Para `std::for_each`, puedes usar una lambda como función.

Ejercicio 5: Contenedores Adaptadores (`std::stack` y `std::queue`)

Crea un programa que demuestre el uso de una pila y una cola:

  1. Pila (`std::stack`):
    • Crea un `std::stack` de caracteres.
    • Añade los caracteres 'A', 'B', 'C' en ese orden.
    • Mientras la pila no esté vacía, saca (`pop`) el elemento superior (`top`) e imprímelo. Observa el orden de salida.
  2. Cola (`std::queue`):
    • Crea un `std::queue` de enteros.
    • Añade los números 10, 20, 30 en ese orden.
    • Mientras la cola no esté vacía, saca (`pop`) el elemento del frente (`front`) e imprímelo. Observa el orden de salida.
Pista (si necesitas ayuda)

Incluye `` y ``. Usa `empty()` para verificar si están vacíos antes de acceder a `top()`/`front()` o `pop()`.
