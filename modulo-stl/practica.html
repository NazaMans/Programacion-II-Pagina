Simulación de Actividades

Estas actividades se centran en la manipulación de estructuras de datos y la interacción con los widgets de Qt.

Ejercicio 1: Mostrar Array

  • Consigna: Crear un programa con un formulario que permita al usuario ingresar 5 números en un array y mostrarlos en un QLabel al presionar un botón.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QLabel.
  • Puntos que debe cumplir:
  • El formulario debe tener 5 QLineEdit.
  • Un botón con el texto "Mostrar Array".
  • Un QLabel para mostrar el contenido del array.
  • Los valores deben almacenarse en un std::array<int, 5> y luego mostrarse.

Ejercicio 2: Vector Dinámico

  • Consigna: Crear un programa que permita al usuario agregar números a un vector dinámico y mostrarlos.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QLabel.
  • Puntos que debe cumplir:
  • Un botón "Agregar al Vector" añade el número ingresado.
  • Un botón "Mostrar Vector" muestra el contenido en un QLabel.
  • Los datos se almacenan en un std::vector<int>.

Ejercicio 3: Manejo de Deque

  • Consigna: Crear un programa donde el usuario pueda agregar números al frente o al final de un deque y mostrarlos en pantalla.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QLabel.
  • Puntos que debe cumplir:
  • Un botón "Agregar al Frente" y otro "Agregar al Final".
  • Un botón "Mostrar deque" muestra los valores en un QLabel.
  • Los números se almacenan en un std::deque<int>.

Ejercicio 4: Lista de Enteros

  • Consigna: Permitir al usuario agregar números al principio o al final de una lista y mostrarlos.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QLabel.
  • Puntos que debe cumplir:
  • Botones "Agregar al Frente" y "Agregar al Final".
  • Un botón "Mostrar Lista" muestra el contenido en el QLabel.
  • Los valores se guardan en un std::list<int>.

Ejercicio 5: Pila de Números

  • Consigna: Crear un programa donde el usuario pueda apilar números y ver cuál es el número en la cima.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QLabel.
  • Puntos que debe cumplir:
  • Botón "Apilar Número" que añade un número a la pila.
  • Botón "Ver Cima" que muestra el número en la cima.
  • La estructura de datos usada es std::stack<int>.

Ejercicio 6: Lista de Tareas

  • Consigna: Crear una aplicación de lista de tareas usando std::vector para almacenar tareas, con opción de agregar y eliminar.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QListWidget.
  • Puntos que debe cumplir:
  • Un QLineEdit y botón "Agregar" para insertar tareas.
  • Un botón "Eliminar" que quita la tarea seleccionada.
  • Las tareas se muestran en un QListWidget y se gestionan en un std::vector<std::string>.

Ejercicio 7: Ordenar Vector y Mostrar Mayor

  • Consigna: Crear una aplicación que permita al usuario ingresar números en un vector, ordenarlos y mostrar el mayor.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QLabel.
  • Puntos que debe cumplir:
  • Un botón "Agregar al Vector" para añadir números.
  • Un botón "Ordenar y Mostrar Mayor".
  • Un QLabel para mostrar el vector ordenado.
  • Un QLabel adicional para mostrar el número más alto.
  • Uso de std::sort para ordenar el vector.

Ejercicio 8: Búsqueda en Vector

  • Consigna: Crear una aplicación que permita al usuario agregar números a un vector y buscar valores dentro de él.
  • Elementos de Qt a utilizar: QTextEdit, QLineEdit, QPushButton, QLabel.
  • Puntos que debe cumplir:
  • Un botón "Agregar al Vector" añade números.
  • Un QTextEdit muestra todos los números ingresados.
  • Un botón "Buscar en el Vector" que verifica si existe un número ingresado.
  • El resultado de la búsqueda se muestra en un QLabel (encontrado o no).
  • Uso de std::find para la búsqueda.

Ejercicio 9: Lista de Palabras Invertida

  • Consigna: Crear una aplicación que permita ingresar varias palabras en una lista, invertir el orden y mostrar la lista invertida.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QTextEdit.
  • Puntos que debe cumplir:
  • Un botón "Agregar Palabra" añade palabras a la lista.
  • Un botón "Invertir Lista" invierte el orden.
  • El resultado se muestra en un QTextEdit.
  • Uso de std::list<std::string> y std::reverse.

Ejercicio 10: Contar Ocurrencias en Lista de Palabras

  • Consigna: El usuario puede agregar palabras a una lista y luego buscar cuántas veces una palabra específica aparece en ella.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QLabel.
  • Puntos que debe cumplir:
  • Un QLineEdit para ingresar una palabra y un botón "Agregar Palabra".
  • Un QLineEdit adicional para ingresar la palabra a contar.
  • Un botón "Contar Ocurrencias" que muestra el resultado en un QLabel.
  • Uso de std::count para calcular cuántas veces aparece la palabra buscada.
