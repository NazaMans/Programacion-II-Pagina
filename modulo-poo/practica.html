Práctica: Ejercicios de Fundamentos de POO

Es el momento de poner en práctica los principios básicos de la Programación Orientada a Objetos. Los siguientes ejercicios te ayudarán a diseñar e implementar clases, trabajar con objetos y entender el encapsulamiento, constructores y destructores.

Asegúrate de compilar y ejecutar tus soluciones en Qt Creator.

Ejercicio 1: Diseño de Clases Simples

Para cada uno de los siguientes conceptos, identifica posibles atributos y métodos. No necesitas escribir código, solo el diseño (nombre de la clase, atributos y métodos, con su posible visibilidad):

  1. Libro: Piensa en las características de un libro y qué acciones se pueden realizar con él.
  2. Estudiante: Qué información describiría a un estudiante y qué acciones podría realizar.
  3. Producto: Cómo describirías un producto en un inventario y qué operaciones se podrían hacer con él.
Ejemplo de diseño para una clase "Coche" 

class Coche {
private:
    std::string marca;
    std::string modelo;
    int anio;
    int velocidadActual; // km/h

public:
    // Constructor
    Coche(std::string _marca, std::string _modelo, int _anio);

    // Métodos (comportamiento)
    void acelerar(int incremento);
    void frenar(int decremento);
    void mostrarInfo();

    // Getters
    std::string getMarca();
    std::string getModelo();
    int getAnio();
    int getVelocidadActual();

    // Setters (si es necesario)
    void setVelocidadActual(int velocidad);
};

Ejercicio 2: Implementación de una Clase "Círculo"

Crea un programa en C++ que defina e implemente una clase `Circulo` con las siguientes características:

  1. Atributos privados:
    • `radio` (tipo `double`)
  2. Métodos públicos:
    • Un constructor que reciba el radio como parámetro y lo inicialice.
    • Un método `calcularArea()` que retorne el área del círculo (`pi * radio * radio`).
    • Un método `calcularPerimetro()` que retorne el perímetro del círculo (`2 * pi * radio`).
    • Un método `getRadio()` para obtener el valor actual del radio.
    • Un método `setRadio(double nuevoRadio)` para modificar el radio, asegurándose de que `nuevoRadio` sea un valor positivo.

En la función `main`:

  1. Crea un objeto de la clase `Circulo` con un radio inicial de `5.0`.
  2. Imprime el área y el perímetro de este círculo.
  3. Modifica el radio del círculo a `7.5` usando el método `setRadio`.
  4. Imprime el nuevo área y perímetro.
  5. Intenta establecer un radio negativo y verifica que el método `setRadio` lo maneje correctamente (ej. no lo acepte o muestre un mensaje de error).
Pista (si necesitas ayuda)

Usa `const double PI = 3.14159;` para el valor de Pi. Para el setter, puedes usar un `if` para validar el `nuevoRadio`.

Ejercicio 3: Clases con Múltiples Constructores

Crea una clase llamada `Punto` que represente un punto en un plano 2D. La clase debe tener:

  1. Atributos privados: `x` y `y` (ambos de tipo `double`).
  2. Constructores públicos:
    • Un constructor por defecto que inicialice `x` y `y` a `0.0`.
    • Un constructor que reciba dos parámetros (`double x_val, double y_val`) e inicialice `x` y `y` con esos valores.
  3. Métodos públicos:
    • `mostrarCoordenadas()` que imprima las coordenadas del punto en formato `(x, y)`.

En la función `main`:

  1. Crea un objeto `Punto` usando el constructor por defecto y muestra sus coordenadas.
  2. Crea otro objeto `Punto` con coordenadas `(3.0, 4.0)` y muestra sus coordenadas.

Ejercicio 4: Uso de Destructores y Memoria Dinámica con Objetos

Revisa el concepto de destructor del módulo de teoría. Luego, modifica el Ejercicio 2 (clase `Circulo`) para que el `radio` sea un puntero (`double*`) y se asigne memoria dinámicamente en el constructor y se libere en el destructor.

  1. Modifica la clase `Circulo` del Ejercicio 2 para que el atributo `radio` sea de tipo `double*`.
  2. En el constructor de `Circulo`, usa `new double` para asignar memoria para el radio e inicialízalo con el valor recibido.
  3. Implementa el destructor de `Circulo` para que use `delete` y libere la memoria asignada al radio. Añade un mensaje en el destructor que indique cuando se está llamando (ej. `std::cout << "Destructor de Circulo llamado." << std::endl;`).
  4. Asegúrate de que `getRadio()` y `setRadio()` manejen correctamente el puntero.

En la función `main`:

  1. Crea un objeto `Circulo` en el Stack y observa cuándo se llama a su destructor.
  2. Crea otro objeto `Circulo` en el Heap usando `new` y asegúrate de usar `delete` para liberarlo y ver el mensaje del destructor.
Pista (si necesitas ayuda)

Recuerda el uso de `*puntero` para acceder al valor. Para liberar memoria de un objeto creado con `new`, debes llamar `delete nombre_del_puntero;`.
