Actividades N1

Estas actividades se centran en la manipulación y la interacción con los widgets de Qt.

Ejercicio 1: Visor de Imagen con Botones de Control

  • Consigna: Crea una aplicación donde al hacer clic en un botón, se muestre una imagen, y al hacer clic en otro, la imagen se oculte.
  • Elementos de Qt a utilizar: QPushButton, QLabel.
  • Objetivo: Controlar la visibilidad de una imagen mediante botones.

Ejercicio 2: Selector de Texto y Estilo de Fuente

  • Consigna: El usuario introduce texto en un campo, selecciona un tamaño de fuente de una lista desplegable, y un botón aplica el estilo al texto mostrado en una etiqueta.
  • Elementos de Qt a utilizar: QLineEdit, QComboBox, QPushButton, QLabel.
  • Objetivo: Modificar el texto y el estilo de una etiqueta basándose en la entrada del usuario y la selección de un combo box.

Ejercicio 3: Generador de Notificaciones Variadas

  • Consigna: Diseña una ventana con tres botones. Cada botón, al ser presionado, debe abrir un tipo diferente de QMessageBox.
  • Elementos de Qt a utilizar: QPushButton (varios), QMessageBox (diferentes tipos).
  • Objetivo: Mostrar diferentes tipos de cuadros de mensaje (informativo, advertencia, crítico) al hacer clic en botones.

Ejercicio 4: Control Deslizante para Simular un Valor

  • Consigna: Implementa un QSlider. A medida que el usuario mueve el control, un QLabel adyacente debe mostrar el valor actual del slider.
  • Elementos de Qt a utilizar: QSlider, QLabel.
  • Objetivo: Actualizar dinámicamente el texto de una etiqueta con el valor de un control deslizante.

Ejercicio 5: Simulador de Estado de Reproducción

  • Consigna: Crea tres botones ("Play", "Pause", "Stop") y una etiqueta que muestre el estado actual de un reproductor ficticio.
  • Elementos de Qt a utilizar: QPushButton (3), QLabel.
  • Objetivo: Cambiar el texto de una etiqueta para simular los diferentes estados de un reproductor de medios al interactuar con botones.

Parte 2: Proyectos Combinados (Qt UI + Conceptos Simples de STL/C++)

Estas actividades integran elementos de interfaz de usuario de Qt con el uso básico de contenedores o algoritmos de la Standard Template Library de C++.

Ejercicio 6: Lista de Compras Interactiva (std::vector<std::QString>)

  • Consigna: Permite al usuario agregar y eliminar ítems de una lista de compras, mostrándolos en un QListWidget.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton (2), QListWidget.
  • Concepto STL/C++: std::vector<std::QString>.
  • Objetivo: Gestionar una lista de ítems tanto en una interfaz gráfica (QListWidget) como en una estructura de datos STL (std::vector).

Ejercicio 7: Validador de Contraseña Básico (std::QString)

  • Consigna: Un campo de texto para contraseña y un botón "Verificar" que valida si la contraseña cumple un criterio simple (ej. longitud mínima).
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QLabel.
  • Concepto STL/C++: Manipulación de std::QString (ej. length(), empty()).
  • Objetivo: Validar la entrada de texto en un campo de contraseña usando lógica básica de manipulación de cadenas de C++.

Ejercicio 8: Historial de Páginas Web Sencillo (std::stack<std::QString>)

  • Consigna: Simula un historial de navegación con botones "Visitar" y "Atrás", mostrando la URL actual.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton (2), QLabel.
  • Concepto STL/C++: std::stack<std::QString>.
  • Objetivo: Implementar una funcionalidad de historial de navegación utilizando una pila STL para gestionar las URLs.

Ejercicio 9: Mini Gestor de Contactos (std::map<std::QString, std::QString>)

  • Consigna: Permite guardar contactos (nombre y teléfono) y luego buscar un contacto por nombre.
  • Elementos de Qt a utilizar: QLineEdit (para nombre), QLineEdit (para teléfono), QPushButton (2), QLabel.
  • Concepto STL/C++: std::map<std::QString, std::QString>.
  • Objetivo: Almacenar y buscar pares clave-valor (nombre-teléfono) utilizando un mapa STL y mostrar los resultados en la interfaz.

Ejercicio 10: Calculadora de Promedio de Números (std::vector<double>)

  • Consigna: El usuario ingresa números, los agrega a una lista, y luego puede calcular su promedio.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton (2), QLabel, QTextEdit.
  • Concepto STL/C++: std::vector<double>, suma de elementos (con bucle o std::accumulate).
  • Objetivo: Recopilar una serie de números y calcular su promedio, mostrando tanto la lista de números como el resultado.
