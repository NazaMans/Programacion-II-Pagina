Parte 1: Proyectos Simples de Interfaz de Usuario (Qt UI Enfocado)

Estas actividades se centran en la manipulación y la interacción con los widgets de Qt.

Ejercicio 1: Conversor de Unidades Simple (Temperatura)

  • Consigna: Crea una aplicación que convierta una temperatura de Celsius a Fahrenheit y viceversa.
  • Elementos de Qt a utilizar: QLineEdit, QRadioButton (o QComboBox), QPushButton, QLabel.
  • Objetivo: Implementar la lógica de conversión de unidades y actualizar dinámicamente la interfaz.
  • Puntos que debe cumplir:
  • Debe validar que la entrada sea numérica.
  • Debe cambiar la conversión dependiendo de la opción elegida.
  • Debe mostrar el resultado con al menos un decimal.

Ejercicio 2: Contador de Clics

  • Consigna: Cada vez que se hace clic en el botón, la etiqueta debe mostrar el número acumulado de clics.
  • Elementos de Qt a utilizar: QPushButton, QLabel.
  • Objetivo: Manejar eventos de clic y actualizar un contador en la UI.
  • Puntos que debe cumplir:
  • El contador debe iniciar en cero.
  • Debe actualizarse correctamente en cada clic.
  • El QLabel debe estar visible siempre.

Ejercicio 3: Selector de Color (RGB Básico)

  • Consigna: El usuario introduce valores R, G y B. Al presionar un botón, el QLabel cambia su fondo.
  • Elementos de Qt a utilizar: QLineEdit (3), QPushButton, QLabel.
  • Objetivo: Procesar entradas numéricas y aplicar propiedades visuales en Qt.
  • Puntos que debe cumplir:
  • Debe validar que los valores estén entre 0 y 255.
  • Debe mostrar un color distinto al cambiar los valores.
  • No debe aceptar campos vacíos.

Ejercicio 4: Verificador de Número Par/Impar

  • Consigna: Determina si un número ingresado es par o impar y muestra el resultado.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QLabel.
  • Objetivo: Implementar lógica condicional básica y mostrar el resultado.
  • Puntos que debe cumplir:
  • Debe validar que la entrada sea entera.
  • El resultado debe mostrarse de forma clara.
  • Debe funcionar correctamente con números negativos.

Ejercicio 5: Calculadora de Área de Rectángulo

  • Consigna: El usuario ingresa base y altura, y se calcula el área del rectángulo.
  • Elementos de Qt a utilizar: QLineEdit (2), QPushButton, QLabel.
  • Objetivo: Realizar operaciones aritméticas y mostrar el resultado.
  • Puntos que debe cumplir:
  • Ambos campos deben contener números válidos.
  • Debe mostrar el resultado con al menos 2 decimales.
  • Debe evitar errores si un campo está vacío.

Ejercicio 6: Generador de Saludos Personalizados

  • Consigna: Al introducir un nombre, muestra un saludo personalizado.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QLabel.
  • Concepto C++: Concatenación de std::string.
  • Objetivo: Combinar cadenas de texto para crear mensajes dinámicos.
  • Puntos que debe cumplir:
  • No debe saludar si el campo está vacío.
  • El saludo debe contener el nombre tal como fue ingresado.
  • Debe respetar mayúsculas y minúsculas.

Ejercicio 7: Visor de Elementos Aleatorios de una Lista

  • Consigna: Muestra un elemento aleatorio de una lista predefinida en un QLabel.
  • Elementos de Qt a utilizar: QPushButton, QLabel.
  • Concepto STL/C++: std::vector, random.
  • Objetivo: Mostrar elementos aleatorios de un contenedor STL.
  • Puntos que debe cumplir:
  • Debe mostrar un nuevo valor cada vez que se presiona el botón.
  • No debe mostrar valores vacíos ni errores.
  • El QLabel debe actualizarse en tiempo real.

Ejercicio 8: Inversor de Cadena de Texto

  • Consigna: Invierte la cadena ingresada y la muestra en un QLabel.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QLabel.
  • Concepto C++: std::reverse o bucle sobre std::string.
  • Objetivo: Manipular texto para invertir su contenido.
  • Puntos que debe cumplir:
  • No debe permitir cadenas vacías.
  • Debe preservar espacios y signos.
  • El resultado debe verse claramente.

Ejercicio 9: Contador de Vocales/Consonantes

  • Consigna: Cuenta cuántas vocales y consonantes hay en una palabra ingresada.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QLabel (2).
  • Concepto C++: std::string, condicionales, bucles.
  • Objetivo: Analizar y clasificar caracteres de una cadena.
  • Puntos que debe cumplir:
  • Distinguir mayúsculas y minúsculas.
  • Ignorar espacios y signos de puntuación.
  • Mostrar el resultado en dos etiquetas separadas.

Ejercicio 10: Simulador de Semáforo

  • Consigna: Simula un semáforo que cambia de estado con cada clic.
  • Elementos de Qt a utilizar: QLabel (3) o QPushButton (3), QPushButton.
  • Objetivo: Controlar visibilidad o color de luces según el estado del semáforo.
  • Puntos que debe cumplir:
  • El ciclo debe ser rojo → verde → amarillo → rojo.
  • Debe haber solo una luz activa por vez.
  • Debe usarse una sola función de control (slot central).
