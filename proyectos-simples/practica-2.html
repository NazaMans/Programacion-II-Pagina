Actividades N2

Estas actividades se centran en la manipulación y la interacción con los widgets de Qt.

Ejercicio 1: Conversor de Unidades Simple (Temperatura)

  • Consigna: Crea una aplicación que convierta una temperatura de Celsius a Fahrenheit y viceversa.
  • Elementos de Qt a utilizar: QLineEdit, QRadioButton o QComboBox, QPushButton, QLabel.
  • Objetivo: Implementar la lógica de conversión de unidades y actualizar dinámicamente la interfaz.

Ejercicio 2: Contador de Clics

  • Consigna: Cada vez que se hace clic en el botón, la etiqueta debe mostrar el número de veces que se ha presionado.
  • Elementos de Qt a utilizar: QPushButton, QLabel.
  • Objetivo: Manejar eventos de clic y actualizar un contador en la UI.

Ejercicio 3: Selector de Color (RGB Básico)

  • Consigna: Permite al usuario introducir valores R, G y B en campos de texto. Al presionar un botón, un QLabel debe cambiar su color de fondo al color resultante.
  • Elementos de Qt a utilizar: QLineEdit (3), QPushButton, QLabel.
  • Objetivo: Procesar entradas numéricas y aplicar propiedades visuales (colores) en Qt.

Ejercicio 4: Verificador de Número Par/Impar

  • Consigna: El usuario introduce un número entero y un botón determina si el número es par o impar.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QLabel.
  • Objetivo: Implementar una lógica condicional básica y mostrar el resultado en la interfaz.

Ejercicio 5: Calculadora de Área de Rectángulo

  • Consigna: El usuario introduce la base y la altura de un rectángulo. Un botón calcula el área y la muestra.
  • Elementos de Qt a utilizar: QLineEdit (2), QPushButton, QLabel.
  • Objetivo: Realizar operaciones aritméticas con valores de entrada y mostrar el resultado.

Ejercicio 6: Generador de Saludos Personalizados

  • Consigna: El usuario introduce su nombre en un campo. Al hacer clic en un botón, un QLabel muestra un saludo personalizado.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QLabel.
  • Concepto C++: Concatenación de std::string.
  • Objetivo: Combinar cadenas de texto para crear mensajes dinámicos.

Ejercicio 7: Visor de Elementos Aleatorios de una Lista

  • Consigna: Predefine un std::vector de palabras. Un botón muestra una palabra aleatoria de la lista.
  • Elementos de Qt a utilizar: QPushButton, QLabel.
  • Concepto STL/C++: std::vector, generación aleatoria.
  • Objetivo: Acceder a elementos aleatorios y mostrarlos en la interfaz.

Ejercicio 8: Inversor de Cadena de Texto

  • Consigna: El usuario introduce una frase. Un botón invierte la cadena y muestra el resultado.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QLabel.
  • Concepto C++: std::reverse o bucle de inversión.
  • Objetivo: Manipular una cadena de texto para invertir su orden.

Ejercicio 9: Contador de Vocales y Consonantes

  • Consigna: El usuario ingresa una palabra. Un botón cuenta vocales y consonantes y muestra el resultado.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QLabel (2).
  • Concepto C++: Iteración sobre std::string y validación de caracteres.
  • Objetivo: Analizar caracteres en una cadena y mostrar conteos.

Ejercicio 10: Simulador de Semáforo

  • Consigna: Usa tres QLabel o QPushButton para representar un semáforo. Un botón cambia el estado en ciclo (rojo → verde → amarillo → rojo).
  • Elementos de Qt a utilizar: QPushButton, QLabel (3).
  • Objetivo: Simular un estado cíclico y cambiar colores o visibilidad de los widgets.
