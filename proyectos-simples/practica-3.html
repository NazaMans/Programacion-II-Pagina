Actividades N3

Estas actividades integran elementos de lógica, validación y estructuras de datos STL junto a la interfaz gráfica de Qt.

Ejercicio 1: Validador de Edad

  • Consigna: El usuario introduce su edad. Un botón determina si es "Mayor de edad" o "Menor de edad".
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QLabel.
  • Objetivo: Practicar conversión de texto a número y condiciones lógicas.

Ejercicio 2: Calculadora de IVA Simple

  • Consigna: El usuario introduce un precio base. Un botón calcula el IVA (21%) y el precio final.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QLabel (2).
  • Objetivo: Realizar cálculos flotantes y mostrar múltiples resultados.

Ejercicio 3: Alternador de Texto

  • Consigna: Un botón alterna el texto de una etiqueta entre dos frases (ej. "Modo ON" y "Modo OFF").
  • Elementos de Qt a utilizar: QPushButton, QLabel.
  • Objetivo: Implementar un estado booleano y alternar visualmente.

Ejercicio 4: Visor de Fecha y Hora Actual

  • Consigna: Al hacer clic en un botón, se muestra la fecha y hora actual.
  • Elementos de Qt a utilizar: QPushButton, QLabel.
  • Concepto C++: Uso de QDateTime o <chrono>.
  • Objetivo: Obtener y mostrar fecha y hora en tiempo real.

Ejercicio 5: Simulación de Interruptor (QCheckBox)

  • Consigna: Al marcar o desmarcar una casilla, un QLabel indica si está "Activada" o "Desactivada".
  • Elementos de Qt a utilizar: QCheckBox, QLabel.
  • Objetivo: Responder a cambios de estado con QCheckBox.

Ejercicio 6: Sumador de Dos Números (con validación)

  • Consigna: El usuario introduce dos números. Un botón suma y muestra el resultado. Si no son válidos, muestra error.
  • Elementos de Qt a utilizar: QLineEdit (2), QPushButton, QLabel.
  • Concepto C++: std::stoi o QString::toInt, validación básica.
  • Objetivo: Validar entrada y sumar correctamente.

Ejercicio 7: Buscador Básico en Lista

  • Consigna: El usuario introduce un número y la app indica si está en una lista predefinida.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QLabel.
  • Concepto STL/C++: std::vector, búsqueda lineal o std::find.
  • Objetivo: Buscar elementos dentro de contenedores STL.

Ejercicio 8: Generador de Números Aleatorios en Rango

  • Consigna: El usuario define un mínimo y máximo. Un botón genera un número aleatorio en ese rango.
  • Elementos de Qt a utilizar: QLineEdit (2), QPushButton, QLabel.
  • Concepto C++: <random>, generación de números con distribución uniforme.
  • Objetivo: Controlar y generar números aleatorios acotados.

Ejercicio 9: Inversor de Números (como texto)

  • Consigna: El usuario introduce un número. La app lo invierte y lo muestra como texto.
  • Elementos de Qt a utilizar: QLineEdit, QPushButton, QLabel.
  • Concepto C++: Conversión a std::string, std::reverse.
  • Objetivo: Manipular strings numéricos.

Ejercicio 10: Alternar Visibilidad de Elementos

  • Consigna: Un botón "Alternar" cambia la visibilidad de varios QLabel.
  • Elementos de Qt a utilizar: QPushButton, QLabel (varios).
  • Objetivo: Controlar visibilidad dinámica de múltiples widgets.
