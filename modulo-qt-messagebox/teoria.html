Introducción a QMessageBox

En Qt, QMessageBox permite mostrar cuadros de diálogo estándar para informar al usuario, pedir confirmación o mostrar advertencias y errores. Son fáciles de usar y se integran con la interfaz creada en Qt Designer (.ui).

Tipos de QMessageBox

  • QMessageBox::Information: Para mostrar información general.
  • QMessageBox::Warning: Para advertencias o alertas.
  • QMessageBox::Critical: Para errores críticos.
  • QMessageBox::Question: Para solicitar confirmación o decisiones del usuario.

Uso básico en widget.cpp

Supongamos que tenemos un botón llamado pushButtonInfo en nuestro formulario widget.ui. La conexión y el uso de QMessageBox se hace así:


#include "widget.h"
#include "ui_widget.h"
#include <QMessageBox>

Widget::Widget(QWidget *parent)
    : QWidget(parent)
    , ui(new Ui::Widget)
{
    ui->setupUi(this);

    // Conectar el botón a un slot
    connect(ui->pushButtonInfo, &QPushButton::clicked, this, &Widget::mostrarInformacion);
}

// Slot definido en widget.h
void Widget::mostrarInformacion() {
    QMessageBox::information(this, "Información", "Este es un mensaje informativo");
}

Ejemplo con diferentes tipos

Podemos mostrar distintos tipos de mensajes usando QMessageBox:


void Widget::mostrarMensajes() {
    // Información
    QMessageBox::information(this, "Info", "Mensaje de información");

    // Advertencia
    QMessageBox::warning(this, "Advertencia", "Mensaje de advertencia");

    // Error crítico
    QMessageBox::critical(this, "Error", "Mensaje crítico");

}

Resumen

  • QMessageBox permite mostrar información, advertencias, errores o preguntas.
  • Se conecta a botones de la interfaz (.ui) mediante slots en widget.cpp.
